# 🎬 Movie Analytics Dashboard

A modern **Full-Stack Movie Analytics Dashboard** built using **React, TypeScript, Tailwind CSS, Material UI, NestJS, and MongoDB**.

This application allows users to manage movies, view analytics, search and filter movies, visualize data through charts, and interact with a responsive dashboard.

---

## ✨ Features

* 🎬 Movie CRUD Operations (Create, Read, Update, Delete)
* 📊 Analytics Dashboard
* 📈 Interactive Charts
* 🔍 Search Movies
* 🎯 Filter by Genre
* 📑 Pagination
* ⚙️ Settings Page
* 📱 Responsive UI
* 🌐 REST API Integration
* 📄 Swagger API Documentation
* 🗄 MongoDB Database

## 🚀 Installation

### Clone Repository

```bash
git clone <YOUR_GITHUB_REPOSITORY_LINK>
```

### Backend

```bash
cd backend
npm install
npm run start:dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
